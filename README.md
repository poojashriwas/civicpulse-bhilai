1.  **Get API Keys**:
    -   **Google Maps API Key**: Enable "Maps JavaScript API", "Places API", and "Geocoding API".
    -   **Firebase Config**: Create a project in Firebase Console -> Web App -> Get Config.

2.  **Insert Keys**:
    -   Open `index.html`: Find `YOUR_GOOGLE_MAPS_API_KEY` and paste your key.
    -   Open `app.js`: Find `YOUR_FIREBASE_API_KEY` (and other config fields) and paste your details.

3.  **Run**:
    -   Double-click `index.html` to open in your browser.
    -   *Note*: For "Detect Location" to work best, run on a local server (like Live Server) or ensure your browser allows geolocation on file:// protocol (Chrome often blocks file:// geolocation).
